牛顿法

三 30 七月 2014
By wwt

牛顿法是近似求解方程的方法，方法是使用函数的泰勒级数的前面几项来寻找方程的根。在统计学习中，牛顿法（Newton method）和拟牛顿法（quasi Newton method）也是求解无约束最优化问题的常用方法。有收敛速度快的优点。牛顿法是迭代方法，每一步需要求解目标函数的海森矩阵（Hessian matrix）的逆矩阵，计算比较复杂。拟牛顿法通过正定矩阵近似海赛矩阵的逆矩阵或海赛矩阵，简化了这一过程。

牛顿法

目标

考虑无约束最优化问题

$$minf(x)$$

其中\(x^*\)为目标函数的极小点。求出f(x)的极小点 \(x^*\)

首先，我们先了解下泰勒级数与海森矩阵。

基本概念

泰勒级数

若函数f（x）在点的某一临域内具有直到（n+1）阶导数，则在该邻域内f（x）的n阶泰勒公式为 用无限项的级数连加式来表示一个函数，这些相加的项由函数在某一点的导数求得。在该点邻域内f（x）的n阶泰勒公式表示为。 对于二阶泰勒展开，二阶的项可以使用海森矩阵来表示。

牛顿法算法

在中学的数学课本上，我们已经接触过牛顿法，它是一种近似求解方程的方法。

我们先看下求解一元方程的牛顿法。假设要求f(x)=0的根，方法使用f(x)的泰勒级数的前面几项来寻找方程f(x)=0的根。

（1）选择一个接近函数f(x)根的点\(x_0\)，计算\(x_0\)在该点的值\(f(x_0)\)与切线斜率\(f'(x_0)\)；

（2）计算经过点\((x_0, f(x_0))\)且斜率为\(f'(x_0)\)的直线与x轴的交点，即方程\(y=f'(x_0)(x - x_0) - f(x_0)\)与x轴的交点，也就是求如下方程的解：

$$f(x_0) = f'(x_0)(x_0 - x)$$

（3）将所求的点的x坐标记为\(x_1\)，通常\(x_1\)会比\(x_0\)更接近方程的根。我们利用\(x_1\)继续迭代：

$$x_{n+1} = x_n - \frac{f(x_n)}{f'(x_n)}$$

（4）每次迭代，都将更接近最终的根。

举一个例子，求\(f(x) = x^2 - x - 2\)的根，对两边求导的\(f'(x) = 2x - 1\)，假设\(x_0 = 3\)。

$$x_1 = x_0 + \frac{f(x_0)}{f'(x_0)} = 2.2$$
$$x_2 = x_1 + \frac{f(x_1)}{f'(x_1)} = 2.011764705882353$$
$$x_3 = x_2 + \frac{f(x_2)}{f'(x_2)} = 2.00004577706569$$

可以看到\(x_3\)已经非常接近函数的一个根2了。

回到，我们在本文开头，提出的目标，求解的过程与上面的例子类似，所不同的是，这里的x是一个向量，而且求的是函数f(x)导数f'(x)的根。

函数f(x)有极值的必要条件是在极值点处一阶导数为0，即梯度向量为0.特别是当\(H(x^{(k)})\)是正定矩阵时，函数f(x)的极值为极小值。

$$\bigtriangledown f(x)=0$$

假设f(x)具有二阶连续偏导数，则将f(x)在第k次迭代值\(x^{(k)}\)邻域内进行二阶展开。

每次迭代从点\(x^{(k)}\)开始，求目标函数的极小点，作为第k+1次迭代值\(x^{(k+1)}\)。对将f(x)在第k次迭代值\(x^{(k)}\)邻域内进行二阶展开，然后对x求导有：

$$\bigtriangledown f(x)=g_k + H_k(x-x^{(k)})$$

其中\(g_k=\bigtriangledown f(x^{(k)})\)，\(H_k\)是f(x)的海森矩阵在\(x^{(k)}\)的值。 假设\(x^{(k+1)}\)满足：

$$\bigtriangledown f(x^{(k+1)})=0$$

由前面两个式子，可以得到：

$$g_k + H_k(x^{(k+1)}-x^{(k)})=0$$

因此可得

$$x^{(k+1)}=x^{(k)}+p_k \; \; \; 其中H_kp_k=-g_k$$

算法：

输入：目标函数f(x)，梯度\(g(x)=\bigtriangledown f(x)\)矩阵,海森矩阵\(H(x)\)，精度要求\(\varepsilon\)

输出：f(x)的极小值点\(x^*\)

（1）取初始点\(x^{(0)}\)，置k=0

（2）计算\(g_k=\bigtriangledown f(x^{(k)})\)

（3）若\(\left \| g_k \right \| < \varepsilon\)，则停止计算，得近似解\(x^*=x^{(k)}\)

（4）计算\(H_k=H(x^{(k)})\)，并求\(p_k\)

$$p_k=-H_k^{-1}g_k$$

（5）置\(x^{(k+1)}=x^{(k)}+p_k\)

（6）置k=k+1，转（2）

步骤（4）中求\(p_k=-H_k^{-1}g_k\)，先求\(H_k^{-1}\)计算比较复杂，所以有其他改进算法。

拟牛顿法思路

考虑用一个n阶矩阵\(G_k=G(x^{(k)})\)来代替\(H_k^{-1}=H^{-1}(x^{(k)})\)。这就是拟牛顿法的基本想法。

将\(x=x^{(k+1)}\)代入下式：

$$\bigtriangledown f(x)=g_k + H_k(x-x^{(k)})$$

得到

$$g_{k+1} - g_k = H_k(x^{(k+1)}-x^{(k)})$$

记\(y_k=g_{k+1} - g_k, \delta_k = x^{(k+1)}-x^{(k)}\)，则

$$y_k = H_k\delta_k$$

或

$$\delta_k = H_k^{-1}y_k$$

上式就称为拟牛顿条件。

$$x=x^{(k)}+\lambda p_k = x^{(k)}-\lambda H_k^{-1}g_k$$

所以f(x)在\(x^{(k)}\)的泰勒展开式可以近似写成：

$$ f(x) = f(x^{(k)})-\lambda g_k^TH_k^{-1}g_k$$

因为\(H_k^{-1}\)是正定的，当\(\lambda\)为一个充分小的正数时，总有\(f(x)< f(x^{(k)})\)，也就是说\(p_k\)是下降方向。

拟牛顿法将\(G_k\)作为\(H_k^{-1}\)的近似，要求矩阵\(G_k\)满足同样的条件。首先，\(G_k\)是正定的，同时\(G_k\)满足下面的拟牛顿条件：

$$\delta_k = G_{k+1}y_k$$

按照拟牛顿条件选择将\(G_k\)作为\(H_k^{-1}\)的近似或者将\(B_k\)作为\(H_k\)的近似的算法就称为拟牛顿法。

按照拟牛顿条件，在每次迭代中可以选择更新矩阵\(G_{k+1}\)：

$$G_{k+1} = G_{k} + \Delta G_{k} $$

这种选择有一定的灵活性，因此有多种具体实现方法。

  • DFP算法：选择\(G_{k+1}\)的方法是，假设每一步迭代中矩阵\(G_{k+1}\)是由\(G_{k}加上两个附加项构成\)
  • BFGS算法: 最流行的拟牛顿算法。BFGS算法与DFP算法类似，只是采用的B来近似H
  • Broyden类算法

BFGS的详细算法：

这时相应的拟牛顿条件是

$$B_{k+1}\delta_k = y_k$$

令

$$B_{k+1}=B_{k}+P_{k}+Q_{k}$$

找到适合条件的\(P_{k}和Q_{k}\)可以求得\(B_{k+1}\).

输入：目标函数f(x)，梯度\(g(x)=\bigtriangledown f(x)\)矩阵，精度要求\(\varepsilon\) （不需要输入海森矩阵）

输出：f(x)的极小值点\(x^*\)

（1）取初始点\(x^{(0)}\)，取\(B_0\)为正定对称矩阵，置k=0

（2）计算\(g_k=\bigtriangledown f(x^{(k)})\)

（3）若\(|\left \| g_k \right \|| < \varepsilon\)，则停止计算，得近似解\(x^*=x^{(k)}\) ，否则转向（4）

（4）计算\(p_k\)

$$p_k=-B_k^{-1}g_k$$

（5）一维搜索：求\(\lambda_k\)使得

$$f(x^{(k)}+\lambda_k p_k) = min_{\lambda \geq 0}f(x^{(k)}+\lambda_k p_k)$$

（6）置\(x^{(k+1)}=x^{(k)}+\lambda_k p_k\)

（7）计算\(g_{k+1} =g(x^{(k+1)})\)

（8）若\(|\left \| g_{k+1} \right \|| < \varepsilon\)，则停止计算，得近似解\(x^*=x^{(k+1)}\) ，否则计算出\(B_{k+1}\)

（9）置k=k+1，转（4）
