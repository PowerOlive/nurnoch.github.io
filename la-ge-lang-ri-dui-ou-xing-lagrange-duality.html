拉格朗日对偶性（Lagrange Duality）

Wed 30 July 2014
By wwt

在求解约束优化问题的时候，常常利用拉格朗日对偶性（Lagrange Duality）将原始问题转化为对偶问题，通过求解对偶问题而得到原始问题的解。该方法用在许多统计学习方法中，如最大熵模型和SVM等。

原始问题

原始最优化问题

考虑最优化问题：

$$min_x f(x)$$
$$s.t. c_i(x) \leq 0 \:\:\:\:\: i=1,2,...,k$$
$$h_j(x) = 0 \:\:\:\:\: j=1,2,...,l$$

称此约束最优化问题为原始最优化问题或原始问题。

广义拉格朗日函数

为了解决原始问题，我们先引入广义拉格朗日（generalized Lagrangian）函数：

$$L(x, \alpha, \beta) = f(x) + \sum_{i=1}^{k}\alpha_i c_i(x) + \sum_{j=1}^{l}\beta_j h_j(x)$$

这里的\(\alpha_i\)和\(\beta_j\)称为拉格朗日乘子。

极小极大问题

现在考虑\(L(x, \alpha, \beta)\)的最大值：

$$\theta_p(x) = max_{\alpha, \beta: \alpha_i \geq 0}L(x, \alpha, \beta) $$

上式中的下标p代表“primal（原始问题）”。如果上式中的x不满足原始问题的约束条件（即存在某些i使得\(c_i(x)>0\)或者\(h_i(x) \neq 0\)），那么就有

$$\theta_p(x) = f(x) + \sum_{i=1}^{k}\alpha_i c_i(x) + \sum_{j=1}^{l}\beta_j h_j(x)= \infty$$

因为若\(c_i(x)>0\)，可取\(\alpha_i \rightarrow +\infty\)，或者\(h_i(x) \neq 0\)，可令\(\beta_j h_j(x) \rightarrow +\infty\)，而将其余的\(\alpha_i ，\beta_j\)都取0.

相反的，若满足原始优化问题的约束条件，那么有\(\theta_p(x) = f(x)\)。所以有

$$\theta_p(x) = \left\{\begin{matrix} f(x) \:\:\:\:\: x满足约束条件\\ \infty \:\:\:\:\: 其他 \end{matrix}\right.$$

因此，如果我们考虑极小化问题

$$min_x\theta_p(x) = min_xmax_{\alpha, \beta: \alpha_i \geq 0}L(x, \alpha, \beta) $$

我们可以看到这与原始最优化问题是等价的，称为广义拉格朗日函数的极小极大问题。这样就把原始最优化问题转化为求解拉格朗日函数的极小极大问题。我们定义原始最优化问题的最优值为

$$p^* = min_x\theta_p(x)$$

称为原始问题的值。

对偶问题

定义

$$\theta_D(x) = min_xL(x, \alpha, \beta) $$

上式中的下标D代表“dual（对偶问题）”。与前面定义的原始问题不同的是，原始问题是通过参数\(\alpha, \beta\)进行优化（极大化），而这里定义的问题是通过\(x\)进行优化（极小化）。

对\(\theta_D(x)\)极大化，就可以得到对偶优化问题

$$max_{\alpha, \beta: \alpha_i \geq 0}\theta_D(\alpha, \beta) = max_{\alpha, \beta: \alpha_i \geq 0}min_xL(x, \alpha, \beta) $$

称为广义拉格朗日函数的极大极小问题。

对偶问题与原始问题除了交换式子中max和min的顺序外，是一样的。同样，我也定义对偶优化问题的最优值

$$d^* = max_{\alpha, \beta: \alpha_i \geq 0}\theta_D(\alpha, \beta)$$

称为对偶问题的值。

原始问题和对偶问题的关系

容易得到（max min 小于等于 min max）：

$$d^* = max_{\alpha, \beta: \alpha_i \geq 0}min_xL(x, \alpha, \beta) \leq min_xmax_{\alpha, \beta: \alpha_i \geq 0}L(x, \alpha, \beta)= p^*$$

在某些条件下，原始问题和对偶问题的最优值相等即\(d^* = p^*\)，这时可以用对偶问题替代原始问题。

定理1：考虑原始问题和对偶问题。假设\(f(x)\)和\(c_i(x)\)是凸函数（convex），\(h_j(x)\)是仿射函数（affine，与线性函数类似只是允许增加一个截距b）；并且存在x，对所有的i有\(c_i(x) < 0\). 则存在\(x^* , \alpha^* ,\beta^*\)，使得在\(x^*\)是原始问题的解，\(\alpha^* ,\beta^*\)是对偶问题的解，并且

$$d^* = p^* = L(x^*, \alpha^*, \beta^*)$$

定理2：考虑原始问题和对偶问题。假设\(f(x)\)和\(c_i(x)\)是凸函数（convex），\(h_j(x)\)是仿射函数（affine）；并且存在x，对所有的i有\(c_i(x) < 0\). 则\(x^*\)是原始问题的解，并且\(\alpha^*, \beta^*\)是对偶问题的解的充分必要条件是\(x^* , \alpha^* ,\beta^*\) 满足下面的Karush-Kuhn-Tucker(KKT)条件：

$$\bigtriangledown_x L(x^*, \alpha^*, \beta^*)=0$$
$$\bigtriangledown_\alpha L(x^*, \alpha^*, \beta^*)=0$$
$$\bigtriangledown_\beta L(x^*, \alpha^*, \beta^*)=0$$
$$\alpha^* c_i(x^*) = 0 \:\:\:\:\: i=1,2,...,k$$
$$c_i(x^*) \leq 0 \:\:\:\:\: i=1,2,...,k$$
$$\alpha^* \geq 0 \:\:\:\:\: i=1,2,...,k$$
$$h_j(x^*) = 0 \:\:\:\:\: j=1,2,...,l$$

其中，上面的第4个式子，称为KKT的对偶互补条件，由此条件可得，若\(\alpha^*>0\)那么\(c_i(x^*) = 0\).
