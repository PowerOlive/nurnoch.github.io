[leetcode] Rectangle Area

Mon 08 June 2015
By wwt

题目

Rectangle Area

Find the total area covered by two rectilinear rectangles in a 2D plane.

Each rectangle is defined by its bottom left corner and top right corner as shown in the figure.

Assume that the total area is never beyond the maximum possible value of int.

分析

分别给定两个矩形的左下角坐标和右上角的坐标，求出两个矩形所占的面积。

  1. 分别计算出两个矩形的面积s1和s2
  2. 两个矩形之和为s，由于两个矩形可能有重叠区域，所以s可能包含了重复的面积
  3. 找到两个矩形重复区域的长和宽
  4. 如果有重复区域，那么就用s减去重复面积

参考代码

class Solution {
public:
    int computeArea(int A, int B, int C, int D, int E, int F, int G, int H) {
        // calculate areas of two separate rects
        int s1 = abs(C - A) * abs(D - B);
        int s2 = abs(G - E) * abs(H - F);

        int s = s1 + s2;

        // intersect of two rects
        int x = min(C, G) - max(A, E);
        int y = min(D, H) - max(B, F);



        if(x > 0 && y > 0)
            return s - x * y;

        return s;
    }
};

给出的test cases有点问题：

Input:  -2, -2, 2, 2, -3, 3, -4, 4
Output: 15
Expected:   17

以上的test case，(GH)在(EF)的左上角，不符合题目的规定（应该在右上角的）。所以计算s1和s2的时候，要加上abs取绝对值，这样就AC了。
