Kaggle 入门指南

五 21 八月 2015
By 往往台

Kaggle是一个非常著名的数据建模和分析的竞赛平台，对于数据挖掘的爱好者来说是一个非常好的学习平台。我们可以通过Kaggle提供的数据集，通过实战提高自己的数据挖掘和分析能力。这篇文章就介绍如何参与一个Kaggle的竞赛，希望能够让更多的数据挖掘爱好者能够熟悉Kaggle竞赛的基本流程，加入到Kaggle中。

我们就以一个简单的竞赛任务——手写数字识别为例，进行介绍。

查看竞赛任务描述

Classify handwritten digits using the famous MNIST data的任务就是训练一个模型，然后输入一个手写数字作为输入，能够判断出它是哪一个数字。

分析数据集

手写字体识别这个比赛，提供了两个数据文件：train.csv和test.csv。这两个文件一个是训练数据集，另外一个是测试数据集。我们需要通过训练数据集生成一个分类器，然后在测试集上进行测试，并且将最终的结果提交给Kaggle。

两个数据文件的内容是0~9数字的手写图像。我们可以先将数据集下载到本地查看数据格式。训练数据集（train.csv）一共有785列，其中第一列为该样本（图像）的标签（label），而其余784列是这个图像的像素值（手写数字的图像为28 * 28）。如下所示。

label pixel0 pixel1 ...
1 0 0 ...
0 0 0 ...
对于测试数据集（test.csv）一共有784列，除了没有lable之外，每个列的含义与训练数据集相同。

测试数据集一共有28000个待分类的手写数字，我们提交分类结果的时候应该按照一行一个结果的格式提交（具体要求的结果格式可以参照页面给的样例），并且需要有一个header表示每列的内容如：

ImageId,Label
1,3
2,5
3,1
剩余的27997个分类结果

分类的结果必须是csv的格式。 最终，通过分类的准确率来衡量你的模型的好坏。

开始比赛吧

1. 编程环境配置

这里我们使用Python作为编程语言，此外我们还需要安装以下几个非常强大有用的Python科学计算和机器学习的库：

  • numpy：提供了大量数组和矩阵计算的函数
  • scipy：Python科学工程计算工具
  • scikit-learn：Python最流行的机器学习库
  • pandas：提供了很多操作数据集的函数

以上的的库都可以通过pip方便的安装。

2. 模型训练

对于手写数字识别的问题，很多模型都可以解决，例如：logistic归回、SVM以及随机森林等。这里我们选用随机森林作为分类器，随机森林虽然稍微有点复杂，但是通常效果还不错，而且利用scikit-learn可以非常方便的实现。 很多人可能对随机森林不太熟悉，这里我还是简单的介绍一下。随机森林（random forest）就是以随机的方式生成一个森林，这个森林是由很多棵决策树构成的。当有一个新的输入的时候，就让每棵决策树去分类，然后查看哪个类被选择的最多，这个输入就属于哪个类。 下面我们就利用scikit-learn来实现随机森林：

#coding=utf-8
from sklearn.ensemble import RandomForestClassifier
from numpy import genfromtxt, savetxt

def main():
    # 创建训练集
    dataset = genfromtxt(open('train.csv', 'r'), delimiter = ',', dtype = 'f8')[1:]
    target = [x[0] for x in dataset] # 第一列为label
    train = [x[1:] for x in dataset]

    # 创建测试集
    test = genfromtxt(open('test.csv', 'r'), delimiter = ',', dtype = 'f8')[1:]

    # 创建并且训练一个随机森林模型
    rf = RandomForestClassifier(n_estimators = 100)
    rf.fit(train, target)
    predicted_results = [[index + 1, x] for index, x in enumerate(rf.predict(test))]

    # 利用随机森林对测试集进行预测，并将结果保存到输出文件中
    savetxt('result.csv', predicted_results, delimiter = ',', fmt = '%d,%d', 
            header = 'ImageId,Label', comments = '')

if __name__ == '__main__':
    main()

首先，分别从两个文件创建训练集和数据集；然后，利用训练集训练一个随机森林模型，直接调用sklearn.ensemble.RandomForestClassifier传入的参数100表示森林有100棵树；最后输出结果的格式为(ImageId,Label)，写出到文件result.csv中。

3. 提交结果

既然已经得到了预测的结果，那么就将结果上传到Kaggle。如果上传格式不对，就无法提交成功，所以要确保按照指定的要求提交。我们将result.csv文件提交，经过判定就得出了模型的准确率和排名，如下所示，排名为395，分数（准确率为0.96457）。

395 new nurnoch 0.96457 1   Fri, 21 Aug 2015 11:10:40

几行的代码就可以取得0.96的准确率还是挺不错的。

以上就是参加Kaggle竞赛的一个简单入门教程。Have fun!
