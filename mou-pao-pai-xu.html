冒泡排序

Wed 13 May 2015
By wwt

冒泡排序算法比较简单，假设待排序列的长度为n，其思想是：

从后往前（或者从前往后）两两比较相邻元素的值，若为逆序（即前一个大于后一个），则交换它们，直到序列比较完，我们称进行了一趟冒泡排序，结果将最小的元素交换到待排序列的第一个位置。下一趟冒泡排序，第一个元素不再参与比较，待排序列的个数减少1，每趟冒泡排序的结果把子序列中最小元素放到了最终的位置。关键字如同气泡一般逐渐往上漂浮至水面，这就是冒泡排序名字的由来。最多n-1趟就能把所有元素排好序。

代码实现如下：

void bubbleSort(vector<int> &A) {
    int n = A.size();
    for(int i = 0; i < n - 1; ++i) {
        bool flag = false;  // 表示本趟冒泡是否发生了交换的标志
        for(int j = n - 1; j > i; --j) {
            if(A[j] < A[j-1]) {
                swap(A[j], A[j-1]);
                flag = true;
            }   
        }   
        if(flag == false)
            return; //本趟冒泡没有发生元素的交换，说明此时序列已经有序了
    }   
}

冒泡排序最坏和平均时间复杂度均为\(O(n^2)\)。 空间复杂度为O(1)。 并且，冒泡排序是稳定的排序方法。
