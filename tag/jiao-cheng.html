    静态网站

    早在多年之前，我就使用过WordPress + MySQL + Apache在国外的主机上搭建过个人博客。但是，对于个人博客而言，使用WordPress这种动态网站，还需要后台支持各种数据库，就显得过于臃肿。最近几年人们纷纷转向静态博客，加之Github Pages和Dropbox这些地方可以免费的放置这些静态页面，于是各种静态网站生成器便如雨后春笋出现，不完全统计的就有392种之多，这里由列出了所有的Static site generator 列表及 Github 关注度排名。

    什么是静态网站

    静态的网站全部由静态网页构成，这些网页都是提前生成好的，不需要与后台数据库进行交交互，并且内容不会动态改变。静态网站具有如下特点：

    • 不需要数据库

    • 访问速度更快

    • 基本没有安全性的问题

    • 使写作者更加专注于内容本身

  2. 使Pelican支持LaTex数学公式

    Fri 10 April 2015
    By wwt

    作为一个技术博客，在写文章的时候，肯定需要经常输入各种公式符号。在之前写一篇文章的时候，发现Pelican默认是不支持编辑数学公式的。之前用过一些在线的markdonw编辑器，都是通过MathJax来支持在markdown中书写数学公式，而且用起来非常的方便，语法基本和LaTex一样。于是，就找到了一种简单的使Pelican支持LaTex语法的数学公式，在这里分享给大家。

    如何使Pelican支持渲染数学公式

    最简单的方法就是使用Pelican的一个插件——pelican_plugin-render_math。 这个插件可以通过使用MathJax使pelican拥有渲染数学公式的能力，并且同时支持Markdown 和reStructuredText，用起来也非常的方便！

    安装

    在Pelican的安装目录中找到Pelican的配置文件pelicanconf.py，在文件中增加：

    PLUGIN_PATH = u"pelican-plugins"

PLUGINS = [ "render_math"]

    为了使插件生效，必须保证插件render_path所在的目录是可以访问的。通常，这个插件同其他插件都一起放在pelican-plugins目录中。

    没错，只要这一步你的pelican网站就可以通过mathjax渲染数学公式了！不需要更改任何模板文件。

    Just use and enjoy!
